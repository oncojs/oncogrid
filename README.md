## Usage
See: [API](API.md)

## Requirements

- Install [node.js](http://nodejs.org/download/)
- Install global modules
    ```
    npm install --global gulp-cli
    
    npm install --global bower
    ```

## Setup

- Install local modules
    ```
    npm install
    ```

- Install libraries for testing
    ```
    bower install
    ```

## Building

To build the minified source:
```
$ gulp
```
